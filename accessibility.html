Accessibility Statement

We strive to meet WCAG 2.1 Level AA.

Our Commitment

This website was designed with accessibility in mind: semantic markup, keyboard navigation, adequate contrast, language switching, reduced-motion options, and accessible data visualizations with textual alternatives.

Conformance Status

We aim for WCAG 2.1 Level AA. If you encounter a barrier, please let us know so we can fix it quickly.

Feedback

Report accessibility issues by opening a GitHub issue in the project repository. Describe the page, the problem, and assistive technology or device used.

Known Limitations

  • Older browsers without modern SVG/CSS support may have degraded chart visuals, but the underlying tables remain accessible.
